## UserDirectory
### Sample project

This is a sample Android project incorporating some of the latest Kotlin/Android features, 
and new-to-me technologies including:

- Navigation 3
- Material Design 3 components
- Kotlin 2.3 explicit backing fields
- koin for dependency injection
- ktor for asynchronous API

This project uses the free public API provided by DummyJSON at https://dummyjson.com/docs
